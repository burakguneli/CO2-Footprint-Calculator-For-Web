About

This is a totally unaccurate carbon emission calculator. Help us to make it more accurate: Github

How it works?

Using browser API's, extension gets total transfer size and "guesses" emitted CO2 according to total transfer size of the page.

We are using word "guess" because it's nearly impossible to measure carbon emission just using transfer size data. This extension accepts that 1 MB transfer emits 0.003 kilograms of CO2.

I Want To Reduce My Emission, What Should I Do?

If you are not a web developer, unfortunately there isn't much to do with this extension on your end. You can just use this extesion for fun. Maybe you can go and plant a tree. I don't know, I'm just a developer :)

If you are a web developer, you can calculate how much your app emits CO2 and you can try to reduce transfer size of your app. Also you can always contribute to this project.
